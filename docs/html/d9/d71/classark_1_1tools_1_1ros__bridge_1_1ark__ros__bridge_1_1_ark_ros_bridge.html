◆ spin()

None ark.tools.ros_bridge.ark_ros_bridge.ArkRosBridge.spin ( self)

Runs the node’s main loop, handling ark messages continuously until the node is finished.

The loop calls self._ark.handle() to process incoming messages. If an OSError is encountered, the loop will stop and the node will shut down.
